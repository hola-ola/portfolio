The Smartass App

Smartass is a web application built with Node.js, Express, MongoDB (and mongoose), handlebars and CSS for styling. The project was built together with a fellow Ironhack student, Luca Janssens. Smartass allows you to set your personal goals with respect to SMART method and easily track your progress thanks to visualisation of your achievemenets. Some of the features you can find in Smartass are: authetification, friends invitation management, profile edition, goal set up and update.
